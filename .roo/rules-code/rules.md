# 项目技术栈总结

本项目主要使用了以下技术：

*   **框架**: Hono (`hono`)
*   **运行时**: Cloudflare Workers (`@cloudflare/workers-types`, `wrangler`)
*   **语言**: TypeScript (`tsconfig.json`)
*   **包管理器**: Bun (`bun.lock`)
*   **构建/部署工具**: Wrangler (`wrangler.jsonc`, `package.json` scripts)
*   **JSX**: Hono JSX (`tsconfig.json`)
*   **Orm**: Prisma
*   **数据库** Cloudflare Worker D1
